# Flight Log Manager

A full-stack flight log management platform designed for pilots to manage and organize flight records.

## Technologies

- React
- Node.js
- Express.js
- MongoDB
- Mongoose
- Tailwind CSS
- REST APIs

## Installation

```bash
git clone https://github.com/Ariyan12005/flight-log-manager.git
cd flight-log-manager/backend
npm install
npm run dev
```
